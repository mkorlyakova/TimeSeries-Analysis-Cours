Стационарный ряд отличается постоянством средних значений и их дисперсий. В нестационарном ряде максимум, что можно сделать - отследить его тренд.

Нестационарные временные ряды - это почти все ценовые графики финансовых активов, столь широко знакомые любому трейдеру. Бычий тренд, медвежий тренд, экстремумы. Исключение составляет боковой дрейф курса на каком-то периоде.

В особый класс нестационарных рядов выделяют интегрированные, они же разностно-стационарные или DS-ряды (англ. Difference Stationary).

Коинтегральные ряды¶

Коинтеграция — свойство нескольких нестационарных (интегрированных) временных рядов, заключающееся в существовании некоторой их стационарной линейной комбинации

коинтеграция рядов - возможность получения стационарного ряда через линейную комбинацию двух (или более) нестационарных интегрированных рядов. Главная мысль состоит в том, чтобы, применяя линейную зависимость, получить из нескольких нестационарных интегрированных рядов, стационарную зависимость.

Тест Энгла-Грэнджера на коинтеграцию : последовательность исследования двух временных рядов Xt и Yt на коинтеграцию.

  1. Анализируются на стационарность исходные ряды. Если они стационарны, то на этом все заканчивается. Коинтеграция тут ни к чему. Пусть ряды Xt и Yt нестационарны.

  2. Далее на стационарность тестируется разность первого порядка исходных рядов. Если она нестационарна, то ряды не являются коинтегральными. В том случае, когда разности первого порядка стационарны и получаются интегрированные ряды первого порядка I(1) (нестационарными рядами со стационарными приращениями - DT(1)) процесс продолжается и переходит к пункту 3.

  3. Составляется линейная комбинация: yt-a xt + ct + b=εt

где:

  • εt - итог линейной комбинации, остаток,
  • с - коэффициент линейного тренда,
  • а - коэффициент коинтеграции xt c yt
  • b - свободный член
  • t - время.
  • учитывает связь только двух временных рядов.

  1. Конечной целью является решение задачи о стационарности ряда εt. “а” подбирают так, чтобы значения εt максимально близко лежали вокруг среднего нулевого уровня, другими словами, обладали минимальным разбросом (дисперсией) относительно него по методу наименьших квадратов (МНК) с Q(a) = min∑εt^2=∑(yt-axt)^2

  2. Подобрав оптимальный коэффициент “а”, ряд εt проверяют на стационарность ( тест Дики-Фуллера).

Если ряд εt - стационарный, вида I(0), то имеет место коинтеграция исходных нестационарных рядов (вида I(1)) Xt и Yt.

Для анализа коинтеграции трех и более рядов более уместен тест Йохансена.

# VAR пример
from statsmodels.tsa.vector_ar.var_model import VAR

# делаем данные
N = 1000  #  общее число точек
Nt = 100 #  точек для теста
data = []
for i in range(N):
    v1 = 2*np.sin(i/10) + np.random.randn()
    v2 = v1 + np.random.randn()
    row = [v1, v2]
    data.append(row)
# учим модель
data = np.array(data)
model = VAR(data[:-Nt,:]) # создали
model.select_order(15)
model_fit = model.fit(maxlags=15, ic='aic') # обучили 
# 
model_fit.summary() # предсказали 100 шагов вперед
